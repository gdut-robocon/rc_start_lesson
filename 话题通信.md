# 话题通信
流程：
1. 按固定格式创建msg
2. 配置有关文件
3. 编译并执行文件



对于这个提交的节点的补充：
这里编写了一个将小写字母转换为大写的节点，使用的时候在终端手动输入一下
```
rostopic pub input_topic std_msgs/String "data: 'hello world'"

```
然后使用
```
rostopic echo output_topic

```
接收节点输出的结果。
